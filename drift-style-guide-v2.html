01 / PHILOSOPHY
The heart of Drift's design
Every design decision flows from one belief: building habits is a human act, not a performance. The interface should feel like a calm, kind friend — not a scoreboard.
"Design for the person who almost quit. Make them feel seen, not judged."
PRINCIPLE 01
Warm over clinical
Every surface, color, and word should feel like it was made by a human who cares. No corporate polish, no sterile whites, no robotic copy. Cream instead of white. Fraunces instead of sans-serif. "You showed up" instead of "Streak: 0."
PRINCIPLE 02
Progress over perfection
The interface celebrates consistency, not flawlessness. 70% is displayed as an achievement, not a failure. No visual punishment for skipping. Partial completion is valid. Color and copy must reinforce this at every touchpoint.
PRINCIPLE 03
Fast over feature-rich
The fastest path to value wins. Check-in in one tap. Add a habit in 10 seconds. Every additional step is a dropped user. Complexity is a design failure. If it takes more than 3 taps, rethink it.
PRINCIPLE 04
Insight over data
Don't show raw numbers. Show meaning. "You always do this on Thursdays" beats "23 completions." Translate data into language a human would actually find useful and emotionally resonant.
PRINCIPLE 05
Delight in restraint
Micro-interactions exist to confirm, not to entertain. One well-placed animation that confirms a check-in is worth more than a confetti explosion. Earn every animation. Make every motion meaningful.
PRINCIPLE 06
Pro without arrogance
The Pro tier must feel premium and warm simultaneously. Gold accents signal value without feeling cold or corporate. Pro features should feel like a gift, not a paywall. The free tier must never feel punished.

14 / VOICE & TONE
Warm. Honest. Human.
Drift speaks like a kind friend who happens to know a lot about habits. Never clinical, never preachy, never over-celebrating. Every word earns its place.
✓ Write like this
"Build habits that actually stick."
"Tomorrow's a fresh start 🌅"
"You showed up 18 out of 20 days. That's real."
"Your journey starts here"
"Start with just one — that's enough."
"Something went wrong. Try refreshing."
Warm · Specific · Human · Short
✕ Never write like this
"Maximize your productivity with AI-powered insights!"
"Streak broken. You've lost your 14-day streak."
"Habit completion rate: 0%. Please complete habits."
"Error 422: Validation failed. Please try again."
"AMAZING JOB!!! You're a HABIT HERO!! 🎉🎉🎉"
Salesy · Punishing · Robotic · Exhausting · Generic
Copy Reference — Key Moments

"Your journey starts here" · "Add one above — it takes 10 seconds"

"🌱 Habit added!"

"✨ All done for today"

"Account created! Please sign in."

"Delete this habit?" · "This cannot be undone."

🎯 Focus Visible — WCAG 2.4.11
Every interactive element must show a focus ring when navigated by keyboard. This is non-negotiable.
/* ✓ Apply to all interactive elements */ :focus-visible { outline: 2px solid var(--sky); outline-offset: 2px; border-radius: inherit; } /* ✕ Never do this */ :focus { outline: none; } /* breaks keyboard nav */

⚡ Reduced Motion — WCAG 2.3.3
Respect user preference. Wrap all decorative animations in this media query.
/* ✓ Wrap all non-essential motion */ @media (prefers-reduced-motion: reduce) { * { animation-duration: 0.01ms !important; transition-duration: 0.01ms !important; } /* Exception: keep progress bar fill transition — it communicates state */ .progress-fill { transition: width 0.3s ease; } }

Never rely on colour alone to communicate errors. Use text + colour + icon together.
